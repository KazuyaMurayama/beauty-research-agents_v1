# beauty-research-agents

美容・アンチエイジング特化マルチエージェントリサーチシステム

## 概要

ユーザーが美容・アンチエイジングに関する質問を1つ入力するだけで、6人の専門家エージェントが並列分析し、統括エディターが統合して最終レポート（Markdown）を自動生成するCLIアプリケーションです。

## セットアップ

```bash
# 依存インストール
pip install -e ".[dev]"

# 環境変数を設定
cp .env.example .env
# .env ファイルにAPIキーを設定してください
```

## 使い方

```bash
python -m src.main
```

プロンプトが表示されたら、美容・アンチエイジングに関する質問を入力してください。

## エージェント構成

| エージェント | 役割 | Web検索 |
|---|---|---|
| 美容皮膚科医 | 医学的観点から回答 | - |
| 美容インフルエンサー | トレンド・実用性 | - |
| 美容家電オタク | デバイス技術分析 | ✓ |
| 論文リサーチャー | 学術的根拠 | ✓ |
| サプリ・医薬品アドバイザー | サプリ・OTC・処方薬 | ✓ |
| 海外トレンドハンター | グローバル情報 | ✓ |
| 統括エディター | 統合・最終レポート | - |
