## 免責事項

本レポートは公表済み臨床研究・RCT・系統的レビューに基づく情報提供を目的としており、医療行為の推奨ではありません。スコアは複数研究の効果量・エビデンスレベルを基にした推計値です。個人差があります。トレチノイン・ヒルドイドは医師処方が必要です。2026年4月時点の情報に基づきます。

## 追加した評価軸について

前回レポートの5軸（シミ・シワ・キメ・美白・毛穴）に、各成分が特に強みを持つ以下5軸を追加した。

| 追加軸 | 追加理由 |
|--------|---------|
| **⑥ニキビ・コメドン** | トレチノインのFDA承認適応症かつ最強エビデンス領域。ナイアシンアミドも抗炎症RCT多数 |
| **⑦赤み・炎症鎮静** | ナイアシンアミドのロサセア治療エビデンス・トラネキサム酸の抗PGE2作用が顕著 |
| **⑧保湿・バリア修復** | ヒルドイドの一次適応症であり「美容効果の底上げ役」としての評価軸として必要 |
| **⑨皮脂・テカリ抑制** | ナイアシンアミドのLevel A専門エビデンスを正当に評価するために追加 |
| **⑩抗酸化・光老化予防** | ビタミンC誘導体の最大の強みであり、紫外線ダメージ予防という独立した効果軸 |

## エグゼクティブサマリー

### 最重要発見 TOP5（拡張版）

1. **トレチノインとナイアシンアミドが「万能型」として際立つ** — 10軸合計でそれぞれ692点・675点。特にトレチノインはニキビ90点が加わり全成分中最高の美容効果を維持
2. **各成分に「突出した得意軸」が明確に存在する** — ビタミンC誘導体の抗酸化92点・ヒルドイドの保湿95点・トラネキサム酸の美白85点は他成分を大きく引き離す「スペシャリスト性能」
3. **ナイアシンアミドは皮脂・テカリ抑制でLevel Aエビデンス（80点）** — 40代男性の皮脂悩みへの科学的根拠が最も高い成分。前回評価で見落とされていた強みが顕在化
4. **ヒルドイドを「保湿役」として再評価すると総合戦略が変わる** — 保湿軸単体で95点（他の4成分は30〜38点）。他成分の浸透・安定化を担う「インフラ成分」
5. **トラネキサム酸の「赤み・炎症鎮静72点」は見落とされがち** — シミ・美白だけでなく肌の炎症抑制にも有効。PGE2阻害機序はロサセア・くすみ原因炎症の抑制に寄与

## 追加5軸の詳細解説

### ⑥ ニキビ・コメドン改善

#### 各成分のスコアと根拠

| 成分 | スコア | EL | 主要根拠・作用機序 |
|------|:------:|:--:|-----------------|
| **トレチノイン 0.025%** | **90** | **A** | 毛包角化正常化（コメドン形成を根本阻止）・皮脂腺萎縮。FDA承認のアクネ治療薬。Kligman 1969, Cunliffe 2001等 多数RCT |
| **ナイアシンアミド 5%** | **58** | B | PGE2阻害・抗炎症。Shalita 1995 RCT（4%ゲルvs クリンダマイシン1%で同等効果）。炎症性ニキビに有効 |
| **トラネキサム酸 1%** | **35** | C | 炎症後の赤み・PIH軽減に間接貢献。コメドン形成への直接作用はなし |
| **ビタミンC誘導体 20%** | **32** | C | 抗菌作用(Propionibacterium acnes)の報告あり。炎症後ニキビ跡の色素沈着改善に有効 |
| **ヒルドイド 0.3%** | **22** | D | ニキビへの直接作用なし。抗炎症効果で炎症後発赤への間接効果のみ |

> **ポイント**: トレチノインのニキビ適応は「医療用効能」として確立しており、美容効果としても10軸中最高の90点を付与。40代男性の大人ニキビ・コメドンに対して最も強力な選択肢。

## 成分別 特化プロファイル（強み・弱みの全体像）

### トレチノイン 0.025% ─ 「万能アクティブの王様。ただし管理が必要」

```
得意軸（70点以上）: シミ85 / シワ90 / キメ88 / 美白75 / 毛穴72 / ニキビ90
弱い軸（50点未満）: 保湿35 / 赤み45 / 安全性48 / 手間40
```
**使い方のコツ**: 最初の4〜8週間のパージングを乗り越えれば最強。ヒルドイド「サンドイッチ法」で刺激を大幅軽減できる。単独では使いにくい → 他4成分との組み合わせが前提。

---

### ナイアシンアミド 5% ─ 「10軸中6軸でTop2圏内の万能型バランサー」

```
得意軸（70点以上）: シミ70 / キメ72 / 美白75 / 毛穴68 / 赤み78 / 保湿72 / 皮脂80
弱い軸（60点未満）: ニキビ58 / 抗酸化42
```
**使い方のコツ**: どのルーティンにも組み込める「接着剤成分」。トレチノインの刺激軽減、ビタミンCの美白作用の補完、ヒルドイドとのバリア強化。単独使いでも十分な効果が期待できる最も取り回しやすい成分。

---

### ビタミンC誘導体 20% ─ 「抗酸化92点の圧倒的スペシャリスト。美白もTop2」

```
得意軸（70点以上）: シミ80 / シワ72 / キメ70 / 美白82 / 抗酸化92
弱い軸（40点未満）: ニキビ32 / 皮脂32 / 保湿38
```
**使い方のコツ**: 朝に使うことで「光老化予防の盾」として最大の効果を発揮。日焼け止めとの相乗効果で光老化を二重に防御。製品の品質（安定性・濃度）が効果に直結するため、信頼できる誘導体製品の選択が重要。

---

### ヒルドイド 0.3% ─ 「保湿95点。5成分の効果を引き出す『インフラ成分』」

```
得意軸（70点以上）: 保湿95 / 安全性96 / 手間93 / ランコスト95
弱い軸（30点未満）: 美白20 / ニキビ22 / 抗酸化18
```
