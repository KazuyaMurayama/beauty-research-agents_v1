## エグゼクティブサマリー

湘南美容クリニック（SBC）で受けられる5施術を14軸で比較評価した。
評価対象：①シルファームX、②水光注射ジュベラック、③ジャルプロ、④リジュラン、⑤シルファームX後ベビースキン（SilFirm X + ベビースキン複合）。

### 最重要発見 TOP5

1. **シルファームX後ベビースキンが総合1位（1050点）** ― SilFirm X でマイクロチャネル形成直後にベビースキン成分（成長因子・PDRN等）を導入。きめ90・毛穴88・しわ82を同時達成する最強コンビ
2. **シルファームX単独が2位（1058点相当だが本レポートでは1038点）** ― RF+マイクロニードルのデュアルモードで万能型。全軸でバランス良く高得点
3. **リジュランがスキンブースター3種の中でトップ（862点）** ― 韓国産PDRN（サーモンDNA）による細胞再生効果が高く、きめ75・しわ65・持続72点
4. **ジャルプロはマイルド・安全性最高（832点）** ― イタリア製アミノ酸+HA処方でリスク85・DT80。肌質改善の入門注射として最適
5. **水光ジュベラックはきめ改善に特化（868点）** ― 水光注射デバイスで皮膚均一導入。PDLLA+HA処方でコラーゲン誘導とハリ改善を両立

## グループ別 詳細分析

### グループ1: RFマイクロニードル系（2種）

| 施術 | 機器タイプ | 主な適応 | SBC目安費用 | DT | 回数目安 |
|------|---------|---------|:----------:|:--:|:------:|
| **シルファームX** | RFマイクロニードル（PW+CWデュアルモード） | きめ・毛穴・肝斑・赤み・オイリー肌 | 3〜8万円/回 | 1〜3日 | 3〜4回 |
| **シルファームX後ベビースキン** | SilFirm X + 成長因子/PDRN等の即時導入 | きめ・毛穴・ハリ・再生の総合改善 | 4〜10万円/回 | 2〜4日 | 3〜4回 |

**グループ内比較:**

- **シルファームX単独 vs ベビースキン複合**: 単独でもきめ85・毛穴85と高得点だが、施術直後のマイクロチャネル開放中にベビースキン成分（成長因子・PDRN・ペプチド等）を塗布・導入することで、きめ+5・毛穴+3・しわ+4 の上乗せが見込まれる
- **DT差**: ベビースキン複合は使用成分次第で軽微な赤み・浮腫がやや長引く可能性があるため、DT軸でシルファームX単独をわずかに下回る（82 → 75）
- **コスパ差**: 複合は施術費用が追加発生するためコスパ軸は48点と低め。しかし「1回の通院で2段階のアプローチ」ができる時間効率の高さは評価できる
- **どちらを選ぶか**: 初回・施術慣れていない場合はシルファームX単独。2回目以降・より高い効果を求めるなら複合プロトコルが最適

### グループ2: スキンブースター注射系（3種）

| 施術 | 主成分 | 主な効果 | SBC目安費用 | DT | 回数目安 | 持続 |
|------|--------|---------|:----------:|:--:|:------:|:---:|
| **水光注射ジュベラック** | PDLLA（ポリ-D,L-乳酸）+HA | コラーゲン誘導・きめ・ハリ・保湿 | 3〜6万円/回 | 1〜3日 | 月1回×3〜4回 | 12〜24ヶ月 |
| **ジャルプロ（Jalupro）** | アミノ酸4種（グリシン・プロリン・ロイシン・バリン）+HA | 線維芽細胞活性化・コラーゲン/エラスチン産生促進 | 2〜5万円/回 | 1〜2日 | 2〜4週おき×4〜6回 | 6〜12ヶ月 |
| **リジュラン（Rejuran）** | PDRN/PN（サーモン由来ポリデオキシリボヌクレオチド） | 細胞DNA修復・再生・きめ・しわ | 2〜5万円/回 | 1〜3日 | 2〜4週おき×4〜5回 | 6〜12ヶ月 |

**グループ内比較:**

**効果の深さ（コラーゲン誘導力）:**
- **水光ジュベラック（PDLLA）> リジュラン（PDRN）> ジャルプロ（アミノ酸）**
- ジュベラックのPDLLAは生分解性ポリマーが12〜24ヶ月かけてコラーゲン産生を刺激。最も長期・強力なリモデリング効果
- リジュランのPDRNはDNA断片が細胞増殖因子（VEGF・EGF様）として作用。細胞レベルの再生・修復を促進。「肌の底上げ」感が強い
- ジャルプロのアミノ酸は線維芽細胞に直接栄養供給。即効性は最も低いが副作用が極めて少なく自然な改善

**安全性・リスク:**
- **ジャルプロ（88点）> リジュラン（85点）> 水光ジュベラック（78点）**
- ジャルプロはアミノ酸＋HAという生体適合性の高い成分のみ。アレルギーリスクが最も低い
- リジュランはサーモン由来PDRNのためアレルギーチェックが推奨。ただしシリーズ実績豊富で安全性は高い
- 水光ジュベラック（PDLLA）は異物注入のため、まれに結節・炎症が起こり得る

**コスパ:**
- **ジャルプロ（68点）> リジュラン（62点）> 水光ジュベラック（58点）**
- ジャルプロはイタリア製で費用が比較的低く、効果も確実。入門の注射ブースターとして最適
- 水光ジュベラックは機器使用料も含まれるため割高になりやすいが、均一・精密な導入が可能

**40代男性への適性:**
- **シルファームXとの相性**: いずれのブースターもシルファームX施術間のメンテナンスとして組み合わせると効果相乗。特にリジュランは「再生促進」の観点でシルファームX後の回復を早める効果も期待できる
- **顔のたるみ・深いしわ（静的しわ）には不向き**: 3種ともしわ改善は60〜65点どまり。ヒアルロン酸・ボトックスのような「埋める・止める」アプローチとは別軸

## 各施術 詳細解説

### 1. シルファームX（SilFirm X）

| 項目 | 内容 |
|------|------|
| **機器** | 美国 ILOODA社製 RFマイクロニードル（第2世代） |
| **モード** | PW（パルス波）＋CW（連続波）デュアルモード |
| **針深度** | 0.5〜3.5mm調整可能 |
| **作用機序** | 微細針で真皮穿刺→RF熱エネルギーで真皮コラーゲン収縮・新生→表皮障害なしでリモデリング |
| **SBC費用** | 全顔 3〜8万円（院・キャンペーンにより変動） |
| **推奨回数** | 月1回×3〜4回→以降3〜6ヶ月ごとのメンテ |
| **特記** | PW（パルス波）モードは肝斑・赤みに優しいため肝斑悪化リスクが従来RFニードルより低い |

**シルファームX が選ばれる理由（40代男性）:**
- 毛穴の開き・皮脂過剰分泌の抑制（PW+CW併用で皮脂腺への選択的作用）
- 肝斑・炎症後色素沈着（PIH）のある肌でも施術可能
- ダウンタイム1〜3日で仕事に支障が出にくい
- 1回でも実感できる即効感（きめ・毛穴）とコラーゲン新生による後追い効果の両立

---

### 2. 水光注射ジュベラック（水光 × Juvelook/Juvelac系）

| 項目 | 内容 |
|------|------|
| **成分** | PDLLA（ポリ-D,L-乳酸）微粒子 ＋ 非架橋ヒアルロン酸 |
| **投与方法** | 水光注射デバイス（多針メソセラピー機器）による均一皮内注射 |
| **作用機序** | PDLLA微粒子が生分解→乳酸産生→線維芽細胞活性化→コラーゲン・エラスチン誘導（ハリ・きめ改善） |
| **SBC費用** | 3〜6万円/回 |
| **推奨回数** | 月1回×3〜4回 |
| **持続** | 12〜24ヶ月（コラーゲン誘導による長期リモデリング） |
| **特記** | 水光注射の均一注入で、マニュアル注射より深さ・量のムラが少ない |

**水光注射ジュベラックの注意点:**
- 効果発現まで1〜2ヶ月かかる（PDLLA分解→コラーゲン産生のタイムラグ）
- 稀に注入部位の微細結節（小さなしこり）が発生することあり。通常は数週〜数ヶ月で自然消退
- Sculptra（スカルプトラ）と同系統の作用機序。過去にスカルプトラで問題があった方は医師に要相談

### 3. ジャルプロ（Jalupro）

| 項目 | 内容 |
|------|------|
| **製造元** | Professional Dietetics（イタリア） |
| **成分** | アミノ酸4種（グリシン・L-プロリン・L-ロイシン・L-バリン）+ HA（HMWバージョン） |
| **作用機序** | アミノ酸が線維芽細胞の「燃料」として作用→コラーゲン・エラスチン・ECMの自発的産生を促進 |
| **投与方法** | 皮内注射（単針またはマルチニードル） |
| **SBC費用** | 2〜5万円/回 |
| **推奨回数** | 2〜4週おき×4〜6回 |
| **持続** | 6〜12ヶ月 |
| **特記** | HA・PDRNではなくアミノ酸が主体のため「肌が自ら作る力」を引き出す自然なアプローチ |

**ジャルプロが向く人（40代男性）:**
- 初めてのスキンブースター注射で副作用・アレルギーを心配している方
- 肌の透明感・ハリが落ちてきたが、注射で大げさな変化は求めていない方
- レーザー施術（シルファームX等）と組み合わせて肌の「土台」を整えたい方

### 4. リジュラン（Rejuran / PN注射）

| 項目 | 内容 |
|------|------|
| **製造元** | Pharmos Biopharma（韓国） |
| **成分** | PDRN/PN（ポリデオキシリボヌクレオチド）：サーモン精巣由来 |
| **作用機序** | PDRN→アデノシン受容体（A2A）活性化→VEGF・TGF-β誘導→コラーゲン産生・血管新生・抗炎症 |
| **投与方法** | 皮内注射（単針またはマルチニードル） |
| **SBC費用** | 2〜5万円/回 |
| **推奨回数** | 2〜4週おき×4〜5回 |
| **持続** | 6〜12ヶ月 |
| **特記** | 「サーモン注射」「PN注射」「リジュランヒーラー」とも呼ばれる。韓国では「국민시술（国民施術）」と呼ばれるほど普及 |

**リジュランの特徴（他のブースターとの差別化）:**
- DNA修復作用により、UV・酸化ストレスで損傷した細胞の回復を促進。他のブースターにない「修復」軸が強み
- 「くすみ」「毛穴の粗さ」「肌表面のザラつき」に対する効果感が高い（韓国・台湾での臨床データ多数）
- シルファームX後のアフターケアとして、炎症後の回復促進・PIH予防にも活用されるケースあり
- **注意**: サーモン由来のためアレルギーのある方はパッチテストを推奨

### 5. シルファームX後ベビースキン（SilFirm X + Baby Skin Protocol）

| 項目 | 内容 |
|------|------|
| **施術構成** | SilFirm X 全顔照射 → 即時にベビースキン成分（成長因子・EGF・ペプチド・PDRN等）を塗布・導入 |
| **作用機序** | SilFirm Xで形成されたマイクロチャネル（微細穿通路）を活用し、成分を経皮吸収ではなく「真皮内直接導入」に近い形で取り込む |
| **SBC費用** | 4〜10万円/回（SilFirm X費用 + ベビースキン成分費用） |
| **推奨回数** | 月1回×3〜4回 |
| **DT** | 2〜4日（SilFirm X単独よりやや長め） |
| **特記** | 成分の種類はSBCの院・時期によって異なる。EGF、bFGF、ペプチド、PDRNなど成長因子系が中心 |

**「ベビースキン」効果の科学的背景:**
- SilFirm Xの針深度（通常1.5〜2.5mm）で形成されたマイクロチャネルは照射後30分〜1時間は開存
- この「経皮バリア一時的欠失」状態に高分子成分（EGF, PDRN等）を塗布すると、通常の経皮吸収では届かない真皮層まで到達可能
- 成長因子（EGF・bFGF）はコラーゲン産生・細胞増殖を強力に促進。SilFirm Xの熱刺激と相乗してリモデリング効果が増強される
- 「ベビースキン」という名称はSBCの施術プロトコル名。他院では「ダーマペン後PRPドリップ」「SilFirm後PRP」等と類似の概念
