## 10. 全テーマ共通の注意事項

| 項目 | 内容 |
|------|------|
| 海外製品の個人輸入 | 健康被害時に医薬品副作用被害救済制度の対象外 |
| 日本未承認成分 | トレチノイン（美容目的）、PDRN注射、メラノタン等は医師相談必須 |
| サプリの過剰摂取 | 亜鉛（銅欠乏）、ビタミンD（高Ca血症）、ビオチン（検査値影響）に注意 |
| エステ vs 医療 | レーザー施術はすべて医療行為。エステサロンでの実施は違法 |
| 韓国渡航医療 | コスパ圧倒的（日本の1/3）だが術後フォロー・法的リスクに注意 |
